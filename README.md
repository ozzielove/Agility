# Agility

A project integrated with TestSprite for AI-powered testing.

## Quick Start

This project is configured to use TestSprite MCP for automated testing through Claude Code.

### Setup

See [TESTSPRITE_SETUP.md](./TESTSPRITE_SETUP.md) for complete setup instructions.

### TestSprite Integration

TestSprite MCP is configured for this project. Once you've set up the MCP server in Claude Code:

- Generate test plans from product requirements
- Create and run automated tests
- View results in the [TestSprite Dashboard](https://www.testsprite.com/dashboard)
